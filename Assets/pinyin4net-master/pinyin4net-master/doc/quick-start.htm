pinyin4net 1.0 Quick Start

This documentation covers the pinyin4net 1.0 release.

If you are new to Chinese, we suggest you begin by reading this article Chinese. And you can read this article for understanding Pinyin system.

A Simple Example

Suppose we want get Hanyu pinyin of Chinese character '李'. 

using System;
using Pinyin4net;

namespace Pinyin4net
{
  public class QuickStart
  {
    public static void Main(string[] args)
    {
      string[] result = PinyinHelper.ToHanyuPinyinStringArray('李');
      foreach (string pinyin in result)
        Console.WriteLine(pinyin);
    }
  }
}

You should get "li3" from the console.

Format Output to Uppercase

using System;
using Pinyin4net;
using Pinyin4net.Format;

namespace Pinyin4net
{
  public class QuickStart
  {
    public static void Main(string[] args)
    {
      HanyuPinyinOutputFormat format = new HanyuPinyinOutputFormat();
      format.CaseType = HanyuPinyinCaseType.UPPERCASE;
      string[] result = PinyinHelper.ToHanyuPinyinStringArray('李',format);
      foreach (string pinyin in result)
        Console.WriteLine(pinyin);
    }
  }
}

You shuld get "LI3" from the console.

Format Output with Tone Options

Get output with tone number, this option is default.

using System;
using Pinyin4net;
using Pinyin4net.Format;

namespace Pinyin4net
{
  public class QuickStart
  {
    public static void Main(string[] args)
    {
      HanyuPinyinOutputFormat format = new HanyuPinyinOutputFormat();
      format.ToneType = HanyuPinyinToneType.WITH_TONE_NUMBER;
      string[] result = PinyinHelper.ToHanyuPinyinStringArray('李',format);
      foreach (string pinyin in result)
        Console.WriteLine(pinyin);
    }
  }
}

You should get "li3" from the console.

Get output with out tone marker, this option must combine with option HanyuPinyinVCharType.WITH_U_UNICODE.

using System;
using Pinyin4net;
using Pinyin4net.Format;

namespace Pinyin4net
{
  public class QuickStart
  {
    public static void Main(string[] args)
    {
      HanyuPinyinOutputFormat format = new HanyuPinyinOutputFormat();
      format.ToneType = HanyuPinyinToneType.WITH_TONE_MARK;
      format.VCharType = HanyuPinyinVCharType.WITH_U_UNICODE;
      string[] result = PinyinHelper.ToHanyuPinyinStringArray('李',format);
      foreach (string pinyin in result)
        Console.WriteLine(pinyin);
    }
  }
}

You should get "lǐ" from the console.

About 'ü' Character

Format output replace 'ü' with 'v', this option is default.

using System;
using Pinyin4net;
using Pinyin4net.Format;

namespace Pinyin4net
{
  public class QuickStart
  {
    public static void Main(string[] args)
    {
      HanyuPinyinOutputFormat format = new HanyuPinyinOutputFormat();
      format.VCharType = HanyuPinyinVCharType.WITH_V;
      string[] result = PinyinHelper.ToHanyuPinyinStringArray('吕',format);
      foreach (string pinyin in result)
        Console.WriteLine(pinyin);
    }
  }
}

You should get "lv3" from the console.
